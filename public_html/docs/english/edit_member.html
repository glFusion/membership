Membership Plugin

Membership Options
Option Description
Plan ID Select the membership plan for this member.
Joined Select or enter the date that this member first joined.
Expires Select or enter the date that this member's membership expires.
Notification Sent? This checkbox indicates that an expiration notification has been sent for this membership. If you wish to suppress notifications, checking this box will prevent a notification from being sent. Note that renewing a membership, either via online purchase or using the "renew" function in the membership list, will reset this value but changing the expiration date manually will not.
Membership Number If membership numbers are used you can enter the member's number here.
Trial Membership? Check this box if this membership is a "trial" membership. A trial membership is exactly the same as a regular membership, but the price of the membership purchase is the "new" price instead of "renewal".
Linked Accounts Any accounts listed here are linked to the current account and to each other to form a "family". You can remove a family member by clicking the red "Delete" icon. Doing so will remove all links to and from that account and any other family members.

This action is carried out immediately. You do not need to save this member&apo;s record unless there are other changes made.
Add Linked Accounts If you'd like to add another account to this member's "family", select the account from the dropdown list. The account will be immediately linked to this account and to all other family members, but it's membership status will not be updated until you click the "Save" button.
Remove this account from family? If you need to remove this account from the family, check this box and click the "Save" button. Do not delete the linked accounts individually as that will remove them from the family rather than removing the current member.
