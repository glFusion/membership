Membership Plugin

Product Edit Form
Option Description
Plan ID Enter the plan ID. This is required and should not normally be changed.
This value will be displayed in several places, so it should be human-readable.
Short Name Enter a short name for this plan. This is required.
Description Enter a description for this plan. This will be shown on the main membership selection page.
Enabled Check this box to enable this plan for purchase; uncheck to disable.
Update Linked Accounts? When a plan is purchased, all accounts that are linked to the purchaser can be updated. If this plan should update linked accounts, check this box.

For example, a "Family" membership may update all linked accounts (family members) while a "Single" membership may only update the purchaser's account.
Purchase Allowed by... Select the glFusion group that is allowed to view and purchase this type of membership. Normally this is probably "All Users" but for special plans, such as free honorary memberships, you may not want the public to be able to purchase them.
Fee Schedule Enter the fees for this membership plan in this table. If the Period Start is "Rolling" (selected in the main plugin configuration), then only a "New" and "Renewal" price need to be entered.

If the Period Start is a specific month, you may wish to pro-rate the membership fee by entering different values for each month in which a membership is purchased.
Online Processing Fee Enter an optional fixed fee to be added to all online membership purchases. This may be the Paypal processing fee if you elect not to include it in the membership fees. Will not be added to payments by check.
Transfer Plan If you need to transfer all members from the current plan to another plan, select the new plan here. This is required before you can delete a plan that has members.
