glFusion Membership Positions

Position
Select the name of this position. Does not need to be unique.
Position Type
Select or enter the type, or category, for this position. Examples: "Board", "Committee".
Once a position type is created by typing in the text field it becomes available for future records as part of the dropdown selection.
Recommendation: use singular names for position lists, e.g. "Officer" instead of "Officers".
Current Member
Select the current officeholder. Select "--- Vacant ---" if the office is currently vacant.
Contact Info (Optional)
Enter the contact information for the position or officeholder. This is the text that will be shown on the group list pages with a link to the contact form. If left blank, the user's email address will be displayed.
Order
Enter a numeric value for the sort order on the group list page.
Enabled?
Normally this should be checked. If you'd like to temporarily disable the display of this position without deleting it, uncheck this box.
Show Vacant?
If this box is checked, "--- Vacant ---" will be displayed when the position is vacant. If unchecked, the position will not be displayed at all when vacant.
>
Include in Lists?
If checked, the position name will be included in profile lists. If multiple positions are held, they are separated by commas. Unchecking this box does not affect whether the member is shown in the lists.
>
Group
Select the site group associated with this position, if any. If selected, users will be added to or removed from this group as they are added to or removed from this position.
